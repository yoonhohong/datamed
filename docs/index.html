Data medicine with R
  • by Yoon H. Hong

임상 데이터 의학 (Clinical Data Medicine with R)

강좌 소개

인공지능과 정밀의료 시대에 접어들면서 의료현장과 의학 연구에서 데이터 분석 능력과 비판적 사고능력은 필수적 역량이 되었다. 본 선택 교과목에서는 강의와 실습을 병행하며, 임상 데이터를 이해하고 분석하기 위한 통계적 추론 및 기계학습을 다룬다. 학생들은 프로그래밍 언어인 R을 활용하여 임상연구 데이터를 다루고 분석 과정을 직접 구현해보는 경험을 하게 된다.
과정은 기초편(4주)과 심화편(4주), 두 단계로 구성된다. 기초편은 프로그래밍 경험이 없는 학생도 수강 가능하며, 데이터 분석을 위한 R의 기본 문법과 데이터 가공 및 시각화, 기초적인 통계적 추론(가설 검정과 모수 추정)을 학습한다. 심화편은 기초편을 수강했거나, 기존에 R 또는 Python 등의 프로그래밍 경험이 있는 학생을 대상으로 하며, 통계적 모델링과 기계학습을 중심으로 심화된 학습을 진행한다.

목표

수강생들은 의학/의료 문제해결을 위한 양적 추론과 데이터 분석, 비판적 사고력 및 협업/소통 역량을 갖추는 것을 목표로 한다.

강의와 실습 계획

전반(기초편)

1주: 데이터 의학
의학과 의료에서 데이터 분석과 양적 추론의 중요성을 이해하고, 데이터 분석을 통한 양적 추론 과정의 전반적 흐름에서 비판적 사고력의 중요성과 기술적, 윤리적 고려사항을 이해한다.

2주: R 기초
R 기본 문법과 작업 환경, 사용법을 익힌다: 데이터 유형과 구조, 데이터 입출력, 프로그래밍(조건문과 반복문, 함수)

3주: 탐색적 데이터 분석
R 패키지를 사용하여 데이터 가공과 시각화를 통해 탐색적 데이터 분석을 할 수 있다.

4주: 통계적 추론
통계적 추론의 기본 개념들을 이해하고, R을 활용하여 가설 검정과 모수 추정을 구현할 수 있다.

후반(심화편)

1주: 회귀 및 분류
선형회귀, 로지스틱회귀를 통해 회귀와 분류 문제에 대한 통계적 모델링을 이해하고, R을 이용해 구현할 수 있다.

2주: 생존 분석
생존분석에 대한 통계적 모델링을 이해하고, R을 이용해 구현할 수 있다.

3주: 지도 기계학습
지도 기계학습의 개념과 주요 알고리즘, 편향과 분산 절충(bias-variance trade-off), 교차검증(cross validation)을 이해하고, 모델 성능을 평가할 수 있다.

4주: 비지도 기계학습
비지도 기계학습의 개념, 주성분분석, 군집분석을 이해하고, 활용할 수 있다.
