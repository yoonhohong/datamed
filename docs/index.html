목표와 내용

  1. 데이터 과학이 의학 연구와 의료의 디지털 전환에 어떻게 활용되고 기여할 수 있는지 이해한다.
  2. R의 기본 문법을 이해하고, 데이터 정제와 가공, 시각화를 할 수 있다.
  3. 협업과 재현가능한 연구(reproducible research)를 위한 데이터 과학의 도구들을 활용할 수 있다(Git/Github, Rmarkdown/Quarto, Shiny).
  4. 기계학습의 기본 개념과 작업 과정을 설명할 수 있다.
  5. 기계학습의 주요 알고리즘을 이해하고, 분류(classification)와 회귀(regression), 비지도 기계학습에 적용할 수 있다.

교재
