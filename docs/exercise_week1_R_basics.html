문제 9

while문을 이용하여 1부터 100까지의 짝수 합을 계산하시오.

해설 9

i <- 1
even_sum <- 0
while (i <= 100) {
  if (i %% 2 == 0) {
    even_sum <- even_sum + i
  }
  i <- i + 1
}
even_sum
## [1] 2550

문제 13

아래 두 코드의 결과를 비교하고, 다른 이유를 설명하시오.

c(TRUE, FALSE, TRUE) &  c(TRUE, TRUE, FALSE)
c(TRUE, FALSE, TRUE) && c(TRUE, TRUE, FALSE)

해설 13

  • 정답:
    • & 결과: c(TRUE, FALSE, FALSE) → element-wise AND
    • && 결과: TRUE → 첫 원소만 평가
  • 교훈: 벡터 비교에는 &/|를 쓰고, 스칼라(첫 원소만) 비교에는 &&/||를 사용.

문제 18

다음 코드에서 에러가 발생하는 이유를 설명하고, 올바르게 수정하시오.

v <- c(10, 20, 30, 40, 50)
v[c(-1, 3)]

해설 18

이유: 양수 인덱스와 음수 인덱스는 동시에 사용할 수 없음.

v <- c(10, 20, 30, 40, 50)
v[c(3)]          # 3번째만
# 또는
v[-1][2]         # 1 제거 후 두 번째(= 원래 3번째)
