1. R 설치

R은 “통계와 데이터 분석을 위한 프로그래밍 언어” 입니다.
먼저 공식 웹사이트에서 R을 설치합니다.

R 다운로드 및 설치

R은 계산기로도 활용 가능합니다.

2 + 2
3^2
10 %% 3

2. 변수와 연산자

my_var <- 4  # 또는 my_var = 4
  • <-는 할당 연산자(assignment operator)입니다.
  • 산술 연산자 예시:

RStudio 다운로드 및 설치

5. 데이터 구조 (Data Structures)

Vector

my_vec <- c(1, 2, 3)
my_vec[2]  # indexing

Factor

f <- factor(c("low", "medium", "high", "low"))
levels(f)

Matrix

m <- matrix(1:6, nrow=2)
m[1, 2]

List

lst <- list(name = "Kim", age = 30, scores = c(90, 95))
lst$age

Data Frame

df <- data.frame(name = c("Kim", "Lee"), age = c(30, 40))
df[1, ]

6. 반복문과 조건문

for loop

sum_val <- 0
for (i in 1:10) {
  sum_val <- sum_val + i
}
sum_val

while loop

i <- 1
while (i <= 5) {
  print(i)
  i <- i + 1
}

조건문

i <- 3
if (i > 2) {
  print("크다")
} else {
  print("작다")
}

7. 함수

내장 함수

x <- c(1, 3, 5, 7)

사용자 정의 함수

square <- function(x) {
  return(x^2)
}
square(4)
